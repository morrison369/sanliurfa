# Arşiv İndeksi

Arşiv klasörlerini aktif teslimat kararı için değil, tarihsel kayıtlar için kullan.

## Yollar
- `docs/archive/cleanup/`: tarihli cleanup, dirty-worktree ve doğrulama kayıtları
- `docs/archive/migration/`: tamamlanmış framework migration geçmişi ve kapanış kayıtları
- `docs/archive/phases/`: legacy faz ve özet referansları

## Kural
- Bir dosya yalnızca geçmiş bir cleanup geçişini veya tarihli doğrulama koşusunu açıklıyorsa arşive alınmalıdır.
- Aktif operasyon politikası root veya üst seviye `docs/` altında kalmalıdır.
