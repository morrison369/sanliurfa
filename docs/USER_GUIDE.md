# Şanlıurfa.com - Kullanım Kılavuzu

## 📱 Mobil Uygulama Kullanımı

### Giriş / Kayıt Olma
1. Uygulamayı açın
2. "Giriş Yap" veya "Kayıt Ol" butonuna tıklayın
3. Email ve şifrenizi girin
4. Veya sosyal medya hesaplarıyla giriş yapın

### Mekan Arama
1. Ana sayfadan arama çubuğuna dokunun
2. Mekan adı veya kategori yazın
3. Filtreleri kullanarak sonuçları daraltın

### Favorilerim
1. Alt menüden "Favorilerim" sekmesine gidin
2. Kaydettiğiniz mekanları görüntüleyin

## 💻 Web Sitesi Kullanımı

### Ana Sayfa
- **Hero Section**: Öne çıkan mekanlar
- **Kategoriler**: Hızlı kategori erişimi
- **Yakınımdaki Mekanlar**: Konumunuza göre öneriler

### Arama Filtreleri
- Anahtar Kelime, Konum, Kategori
- Fiyat Aralığı, Puan
- Özellikler (WiFi, Otopark)
