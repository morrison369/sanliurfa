# Weekly Frontend Quality Rhythm

## Haftalık Döngü
1. Tasarım drift taraması (`design:drift:gate`)
2. Görsel alaka taraması (`content:image:relevance:gate`)
3. Tipografi/kontrast spot kontrolü (landing + 3 kritik sayfa)
4. Screenshot diff kontrolü
5. Kapanış notu: düzeltilenler / kalan riskler

## Amaç
- Tema bütünlüğünü kalıcı korumak
- Alakasız görsel ve stil sapmalarını erken yakalamak
- Prod’a çıkmadan önce net kalite bariyeri sağlamak
