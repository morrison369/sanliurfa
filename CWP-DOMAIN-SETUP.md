# CWP Panel Domain ve SSL Kurulum Rehberi\n\n## 🚨 DNS Durumu\n\n**Mevcut DNS:**\n- `sanliurfa.com` → Cloudflare (172.67.153.245, 104.21.12.251) ❌\n- **Hedef:** `sanliurfa.com` → 168.119.79.238 ✅\n\n---\n\n## 📋 Adım 1: DNS Güncelleme (Cloudflare)\n\n1. **Cloudflare Panel**'e giriş yap: https://dash.cloudflare.com\n2. **sanliurfa.com** domainini seç\n3. **DNS** sekmesine git\n4. **A kaydını** bul ve düzenle:\n   ```\n   Type: A\n   Name: @ (veya sanliurfa.com)\n   IPv4 address: 168.119.79.238\n   Proxy status: DNS only (gri bulut) ⚠️ ÖNEMLİ!\n   TTL: Auto\n   ```\n5. **Kaydet**\n\n⚠️ **Proxy (turuncu bulut) KAPALI olmalı!** Aksi halde CWP SSL alamaz.\n\n---\n\n## 📋 Adım 2: CWP Panel Domain Ekleme\n\n1. **CWP Panel**'e giriş: https://168.119.79.238:2083\n   - Kullanıcı: `sanliur`\n   - Şifre: `CHANGE_ME_CWP_SSH_PASSWORD`\n\n2. **Sol Menü** → `Domains` → `Add Domain`\n\n3. **Formu doldur:**\n   ```\n   Domain: sanliurfa.com\n   Document Root: /home/sanliur/public_html\n   PHP Version: PHP 8.2 (veya mevcut)\n   ```\n\n4. **Create** butonuna tıkla\n\n---\n\n## 📋 Adım 3: AutoSSL (Let's Encrypt)\n\n1. **CWP Panel** → `Webserver Settings` → `SSL Certificates` → `AutoSSL`\n\n2. **Alanları doldur:**\n   ```\n   User: sanliur\n   Domain: sanliurfa.com\n   ```\n\n3. **Issue Certificate** butonuna tıkla\n\n4. **Sonuç:**\n   - ✅ Sertifika başarıyla oluşturuldu\n   - HTTPS aktif olur\n\n---\n\n## 📋 Adım 4: Apache Proxy Ayarı (CWP)\n\n1. **CWP Panel** → `Webserver Settings` → `Webserver Domain Conf`\n\n2. **Seç:**\n   ```\n   User: sanliur\n   Domain: sanliurfa.com\n   ```\n\n3. **Configuration:**\n   ```\n   Webserver: Apache\n   Proxy: Custom Port\n   Port: 6000\n   IP: 127.0.0.1\n   ```\n\n4. **Rebuild webserver conf for domain on save** ✅ İşaretle\n\n5. **Save Changes**\n\n---\n\n## 📋 Adım 5: WWW Yönlendirme (Opsiyonel)\n\n`www.sanliurfa.com` → `sanliurfa.com` yönlendirme:\n\n1. **CWP Panel** → `Domains` → `sanliurfa.com` (düzenle)\n\n2. **Aliases** ekle:\n   ```\n   www.sanliurfa.com\n   ```\n\n3. **Save**\n\n---\n\n## ✅ Test\n\nDNS güncellemesi 5-60 dakika sürebilir.
