# CWP (CentOS Web Panel) Apache Yapılandırması\n\n## Apache Proxy Ayarları (Port 6000)\n\n### 1. CWP Panel'e Giriş\n- URL: https://168.119.79.238:2083\n- Kullanıcı: sanliur\n- Şifre: CHANGE_ME_CWP_SSH_PASSWORD\n\n### 2. Domain Yapılandırması\n```\nWebserver Settings → Webserver Domain Conf\n```\n\nVeya doğrudan vhost dosyasını düzenleyin:\n\n```bash\n# SSH ile bağlan\nssh sanliur@168.119.79.238 -p 77\n\n# Vhost dosyasını düzenle (dosya yolu değişebilir)\nsudo nano /etc/apache2/conf.d/vhosts/sanliurfa.com.conf\n# veya\nsudo nano /usr/local/apache/conf/extra/httpd-vhosts.conf\n```\n\n### 3. Proxy Ayarları\n```apache\n<VirtualHost *:80>\n    ServerName sanliurfa.com\n    ServerAlias www.sanliurfa.com\n    DocumentRoot /home/sanliur/public_html\n    \n    ProxyPreserveHost On\n    ProxyPass / http://127.0.0.1:6000/\n    ProxyPassReverse / http://127.0.0.1:6000/\n    \n    # WebSocket desteği (gerekirse)\n    RewriteEngine On\n    RewriteCond %{HTTP:Upgrade} websocket [NC]\n    RewriteCond %{HTTP:Connection} upgrade [NC]\n    RewriteRule ^/?(.*) "ws://127.0.0.1:6000/$1" [P,L]\n</VirtualHost>\n\n# HTTPS (SSL)\n<VirtualHost *:443>\n    ServerName sanliurfa.com\n    ServerAlias www.sanliurfa.com\n    DocumentRoot /home/sanliur/public_html\n    \n    SSLEngine on\n    SSLCertificateFile /home/sanliur/public_html/ssl/sanliurfa.com.crt\n    SSLCertificateKeyFile /home/sanliur/public_html/ssl/sanliurfa.com.key\n    \n    ProxyPreserveHost On\n    ProxyPass / http://127.0.0.1:6000/\n    ProxyPassReverse / http://127.0.0.1:6000/\n</VirtualHost>\n```\n\n### 4. Apache Modüllerini Etkinleştir\n```bash\n# Proxy modülleri\nsudo yum install mod_proxy mod_proxy_http mod_proxy_wstunnel\n\n# Apache'yi yeniden başlat\nsudo systemctl restart httpd\n# veya\nsudo service httpd restart\n```\n\n### 5. Test\n```bash\n# Uygulama çalışıyor mu?\ncurl http://127.0.0.1:6000/\n\n# Apache proxy çalışıyor mu?\ncurl http://sanliurfa.com/\n```\n\n## Alternatif: .htaccess ile Proxy\nEğer CWP vhost erişiminiz yoksa `.htaccess` kullanabilirsiniz:\n\n```htaccess\n# /home/sanliur/public_html/.htaccess\nRewriteEngine On\nRewriteRule ^(.*)$ http://127.0.0.1:6000/$1 [P,L]\n```\n\n**Not:** `mod_proxy` gereklidir.\n\n## Cloudflare + CWP\nCloudflare kullanıyorsanız:\n\n1. Cloudflare SSL/TLS modu: **Full (strict)**\n2. CWP'de SSL sertifikası yüklü olmalı\n3. Origin sertifikası kullanabilirsiniz\n\n## Hata Ayıklama\n\n### 502 Bad Gateway\n```bash\n# Node.js uygulaması çalışıyor mu?\nsource ~/.nvm/nvm.sh\npm2 status\n\n# Port dinleniyor mu?\nnetstat -tlnp | grep 6000\n\n# Logları kontrol et\npm2 logs sanliurfa\n```\n\n### 503 Service Unavailable\n```bash\n# Apache proxy modülü yüklü mü?\nhttpd -M | grep proxy\n\n# SELinux kısıtlaması olabilir\nsudo setsebool -P httpd_can_network_connect 1\n```\n\n## Komutlar Özeti\n\n```bash\n# SSH\ncd /home/sanliur/public_html\n\n# Node.js uygulama yönetimi\nsource ~/.nvm/nvm.sh\npm2 list\npm2 logs sanliurfa\npm2 restart sanliurfa\n\n# Apache\nsudo systemctl restart httpd\nsudo systemctl status httpd\nsudo tail -f /var/log/httpd/error_log\n\n# PostgreSQL\npsql -U sanliur_sanliurfa -d sanliur_sanliurfa\n```\n
